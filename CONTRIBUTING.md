# 🤝 Mitmachen bei GuardianHelper

Danke dass du zum Projekt beitragen möchtest!

## Schnellstart

```bash
# 1. Repo forken (GitHub → Fork-Button)
# 2. Fork klonen
git clone https://github.com/DEIN-NAME/guardian-helper-wow.git

# 3. Feature-Branch erstellen
git checkout -b feature/mein-feature

# 4. Änderungen machen & testen
# 5. Committen & PR öffnen
git commit -m "feat: Beschreibung"
git push origin feature/mein-feature
```

## Was wir gerne hätten
- 🐛 Bugfixes (falsche Spell-IDs, Cooldown-Zeiten)
- ✨ Features (Sound-Alerts, Swing-Timer, Themes)
- 🌍 Übersetzungen (Spell-Namen für FR, RU, KO...)
- 📖 Dokumentation

## Code-Stil
- 4 Spaces Einrückung, keine Tabs
- `local` wo immer möglich
- PascalCase für Funktionen, GROSSBUCHSTABEN für Konstanten
- Kommentare auf Englisch

## Commit-Konvention
```
feat:     Neues Feature
fix:      Bugfix
docs:     Dokumentation
refactor: Code-Umbau
chore:    Sonstiges
```

## PR Checkliste
- [ ] Auf Classic Era / TBC Classic getestet
- [ ] Kein Lua-Error beim /reload
- [ ] CHANGELOG.md aktualisiert
